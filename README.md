# myBlog
我的个人Django博客系统
项目的static文件是在网上找的
项目内容：  
1、后台文章管理，包括新增、删除和编辑 
2、后台分类管理，包括新增、删除和编辑 
3、后台标签管理，包括新增、删除和编辑 
4、前台列表页展示文章概要信息、发布时间、 
5、文章分类、标签、浏览次数 
6、前台文章列表页分页展示 
7、前台文章列表页点击”阅读全文”显示文章详细内容
